# AI-Based Multi-Model System for Liver Disease Risk Assessment

This repository presents an AI-based multi-model system designed to assess the risk of various liver diseases using clinical and laboratory data.

## Key Features
- Multi-model architecture
- Risk assessment (not diagnosis)
- Rule-based + Machine Learning integration
- Ethical and clinical considerations included

## Disclaimer
This system is intended for research and educational purposes only and does not provide medical diagnosis.
